# Client Tracker CRM Design System

## Core Theme
- **Theme**: Modern Clean CRM
- **Primary Color**: Indigo `#4f46e5`
- **Secondary Color**: Violet `#7c3aed`
- **Background**: Slate-50 `#f8fafc`
- **Sidebar Background**: Slate-900 `#0f172a`

## Layout Guidelines
- **Sidebar**: Fixed width 64 (`w-64`), dark background with active indigo states. Text labels are always visible in `#94a3b8` (Slate 400) and change to `#ffffff` (White) on hover.
- **Page Container**: Soft background with `animate-fadeIn` on view transitions.
- **Cards**: Large corner radius (`rounded-2xl`), simple light borders (`border-slate-100`), elevated shadows on hover state (`hover:shadow-md`).
- **Modals**: Centered backdrop blur with `bg-slate-900/50`, maximum height restricted to `90vh`, and inner content forms scrollable.
- **Tables**: Clear borders, bold typography for headers, color-coded stage badges.
